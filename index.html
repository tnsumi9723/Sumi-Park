PROJECT 01

Interpark

Book Store (Open API / Json)

POINT COLOR

국내 웹 도서 사이트 인터파크의 클론 사이트입니다.
페이지 내의 도서 내용들은 KaKao에서 제공하는 Open API를 사용하였습니다.

도서사이트의 특징인 많은 정보로 인하여 복잡한 레이아웃을
좀 더 간결하며 모던하게 재해석했습니다.

뿐만 아니라 정적인 느낌을 탈피하고, 더 많은 정보를 제공하고자
jQuery를 활용하여 네비게이션과 슬라이드를 접목시킨 상품 리스트, 게시판 등의
다양한 인터렉션 요소를 배치하였습니다.

도서를 구매할 수 있는 Sub Page도 함께 제작하였습니다.

PROJECT 02

Ecolife

React Website (Redux / Router)

POINT COLOR

친환경 푸드 스토어 Ecolife의 홍보 웹 사이트 입니다.

React Bootstrap의 Nav, Button, Nav Dropdown 등을 사용하여,
다양한 사용자에 맞게 반응하도록 만든 반응형 웹 페이지 입니다.
React-Router-dom을 사용하여 각각에 맞는
Url을 불러올 수 있도록 하였습니다.

친환경 푸드의 싱그러움을 표현하기위해 블루계열의 컬러인
rgb(134, 215, 237)를 포인트 컬러로 사용하였습니다.

카트에 다양한 상품을 담을 수 있고,
로그인 폼을 사용하여 실제 쇼핑몰 처럼 제작하였습니다.

PROJECT 03

Ferrano

Responsive Website

POINT COLOR

레스토랑 FERRANO의 홍보 웹사이트입니다.
다양한 사용자에 맞게 반응하도록 미디어쿼리를 사용한 반응형
웹페이지입니다.

모노톤에 골드를 포인트 컬러로 사용하여 이미지에
집중도를 높였습니다. 레스토랑 예약, 메뉴 가격 ,오픈시간 ,위치 등
다양한 정보를 제공합니다.

서브 페이지는 레스토랑 소식을 블로그 형식으로 제작하여 더욱
상세하게 정보를 제공하고 있습니다.

PROJECT 04

The Sill

Shopping Mall, Json

POINT COLOR

다양한 식물을 판매하는 사이트 The Sill입니다.

흰색 배경을 이용하여 전체적으로 심플한 느낌을 주었습니다.
포인트 컬러인 초록계열(rgb(0, 154, 123)) 색상을 이용하여
식물만의 상쾌하고 맑은 생명력을 표현했습니다.

메인 페이지의 다양한 상품이미지는 Json을 사용하여
데이터 값을 받아와 구현했습니다.

상단 네비게이션 뿐만 아니라 사이트 내 많은 정보를
Footer에서도 쉽세 이동할 수 있도록 Quick Menu를 제공합니다.
